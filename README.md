# AssignmentManagement
C'est la partie Back End du projet MBDS Madagascar 2021 - 2022.

Voici le lien du projet déployé : https://back-projet-buffa.herokuapp.com/

## Pré-requis

- Angular CLI version '13.2.5'
- IDE : Visual Studio Code ( par exemple )
- Node version - '16.14.0'

## Installation

Pour tester le projet localement :
- Clonez le [Projet](https://github.com/AndyRamanantoanina/ProjetMrBuffet_Back.git) 
- La base de données est déjà sur le cloud.
- Installez les packages par ``npm install``
- Démarrez le projet par ``npm start``
- Vous pouvez testez l'API sur : localhost:8010/api/
